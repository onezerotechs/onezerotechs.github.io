AWS Services

Alguns serviços AWS descritos no curso Skill Builder:

[Principais serviços]

EC2: Serviço de máquinas virtuais na nuvem.

ECS: Gerenciador de contêineres Docker na AWS.

EKS: Kubernetes gerenciado pela AWS.

SNS: Serviço de envio de notificações em tempo real.

SQS: Fila de mensagens para comunicação assíncrona.

AWS Fargate: Executa contêineres sem precisar gerenciar servidores.

AWS CloudFront: CDN para distribuição rápida de conteúdo.

AWS Lambda: Executa código sob demanda sem servidor.

Região: Conjunto de data centers em uma localização geográfica.

Zona: Subdivisão de uma região, usada para alta disponibilidade.

[Armazenamento]

Amazon EBS: Armazenamento em bloco usado com EC2, ideal para dados persistentes e de alto desempenho.

Amazon S3: Armazenamento de objetos altamente escalável, usado para backups, sites, arquivos e dados.

Tipos do S3: Standard, Intelligent-Tiering, Glacier, etc., variam por custo, frequência de acesso e tempo de recuperação.

Amazon EFS: Sistema de arquivos elástico e compartilhado, ideal para múltiplas instâncias EC2 acessando os mesmos dados.

[Bancos de Dados]

Amazon RDS: Serviço gerenciado para bancos relacionais (MySQL, PostgreSQL, SQL Server, etc.).

Amazon DynamoDB: Banco NoSQL gerenciado e altamente escalável, com latência de milissegundos.

Amazon Redshift: Armazém de dados (data warehouse) rápido e escalável para análises em grande escala.

AWS Database Migration Service (DMS): Migra bancos de dados locais ou em nuvem com pouca ou nenhuma parada.

Amazon Neptune: Banco de dados de grafos para aplicações que lidam com relações complexas.

Amazon Managed Blockchain: Serviço gerenciado para criar e gerenciar redes blockchain.

Amazon QLDB (Quantum Ledger Database): Banco de dados com histórico imutável e verificável de transações.

Amazon ElastiCache: Cache na memória para acelerar aplicações, compatível com Redis e Memcached.

Amazon DAX (DynamoDB Accelerator): Cache in-memory para DynamoDB, reduz latência de milissegundos para microssegundos.

[Segurança]

AWS Shield: Proteção automática contra ataques DDoS.

AWS WAF (Web Application Firewall): Firewall personalizável para bloquear tráfego malicioso em aplicações web.

[Monitoramento]

Amazon CloudWatch: Coleta e monitora métricas, logs e alarmes de recursos AWS.

AWS CloudTrail: Registra atividades de API e ações no console para auditoria.

AWS Trusted Advisor: Fornece recomendações de boas práticas sobre segurança, desempenho e custos.

[Preços e Suporte]

AWS Budgets: Permite definir alertas e limites de gastos e uso.

AWS Cost Explorer: Visualiza e analisa custos e uso da AWS ao longo do tempo.

AWS Support: Oferece planos de suporte técnico e consultivo para clientes.

AWS Marketplace: Loja com softwares prontos para uso em nuvem, oferecidos por terceiros.

[Migração para AWS]

AWS Cloud Adoption Framework (AWS CAF): Framework com boas práticas para ajudar empresas a planejarem e executarem sua migração para a nuvem, considerando áreas como segurança, governança, operações e pessoas.

AWS Snowcone, Snowball e Snowmobile: Dispositivos físicos para migração de grandes volumes de dados para a AWS:

Snowcone:
Pequeno e portátil, ideal para cargas menores

Snowball:
Robusto, usado para dezenas de terabytes

Snowmobile:
Caminhão com data center para exabytes — usado em migrações massivas

VMware Cloud on AWS: Permite rodar ambientes VMware (vSphere, vSAN, NSX) diretamente na AWS, facilitando a migração sem reestruturar apps ou redes.

[Serviços de IA/ML e Análise]

Amazon SageMaker: Plataforma para criar, treinar e implantar modelos de Machine Learning de forma rápida e gerenciada.

Amazon Lex: Serviço de chatbot com reconhecimento de voz e linguagem natural, baseado na mesma tecnologia do Alexa.

Amazon Textract: Extrai automaticamente texto e dados estruturados de documentos digitalizados (como PDFs, formulários, contratos, etc.).

AWS DeepRacer: Mini carro autônomo físico e simulado usado para aprender reforço de Machine Learning. Ideal para fins educativos e competições.

AWS Ground Station: Permite controlar satélites e processar dados de antenas diretamente pela AWS — focado em aplicações espaciais.

Amazon CodeWhisperer: Assistente de programação baseado em IA que sugere código em tempo real, semelhante ao GitHub Copilot.

[Arquitetura]

AWS Well-Architected Framework: Conjunto de princípios e boas práticas para criar sistemas na nuvem bem projetados, avaliando cinco pilares: 1 - Excelência operacional 2 - Segurança 3 - Confiabilidade 4 - Eficiência de performance 5 - Otimização de custos
